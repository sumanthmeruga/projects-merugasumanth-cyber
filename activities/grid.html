answering the questions

Elements in HTML can be inside other elements. When an element holds other elements it is known as the parent, and the other elements contained in the parent element are referred to as children. The parent-child relationship is important when using Flexbox or Grid layout because layout properties, such as display: flex or display: grid apply to the children to alter their layout. Without the parent-child relationship, a layout system cannot document or organize elements.
