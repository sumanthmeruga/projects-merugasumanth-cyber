When would you use a table?

You use a table to present information displayed in both rows and columns such as schedules, prices, or statistics.

When would you NOT use a table?

You would not use a table for page layout or design. You should use CSS for arranging content instead.

When do you need to do extra work for table accessibility?

You do extra work when the table is complex (has multiple headers or sections). You would want to add captions, the appropriate header tags, and scope attributes to assist screen readers in understanding the table.
