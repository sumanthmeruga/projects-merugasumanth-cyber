Positioning

What is normal flow in HTML?

Normal flow refers to how the browser automatically lays out the HTML elements before modifying any of these settings by using CSS.

What does float do? Why do you need to clear the float?

Float is used to shift an element to the left or right within its container, allowing other content to wrap around it.

What do absolute, fixed, and relative do?

figure class="temple hammer">

relative Positions an element relative to its standard position. It will not remove it from the standard flow. absolute Positions an element exactly where you want it relative to the nearest ancestor that has been positioned. It will take it out of the standard flow. fixed Positions an element relative to the browser window (even if you scroll). Fixed positioning is commonly used with nav bars or pop-ups.

How can you make nice navigation buttons for mobile devices?

Use CSS Flexbox or Grid to align your buttons evenly. Use media queries to adjust size and layout on small screens. Use CSS (padding, border-radius, color), to make it look nice.
