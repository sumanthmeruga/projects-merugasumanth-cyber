What is JavaScript? Why is it called JavaScript?

JavaScript is a programming language used primarily to make websites interactive (things like buttons, animations, forms, etc.) It started as mocha, then live-script, and was later renamed JavaScript to get greater marketing appeal, mainly because of the popularity of the language Java during that time and Netscape wanted more developers to consider using it.
